# 📄 AI Resume Screener & ATS Matcher

![Python](https://img.shields.io/badge/Python-3.11-blue)
![Streamlit](https://img.shields.io/badge/Streamlit-Live-red)
![Machine Learning](https://img.shields.io/badge/Machine%20Learning-NLP-green)
![License](https://img.shields.io/badge/License-Educational-yellow)

An AI-powered Resume Screening and ATS (Applicant Tracking System) Matcher built using Machine Learning, NLP, and Streamlit. The application compares resumes with job descriptions, calculates ATS compatibility scores, identifies matching and missing skills, provides hiring recommendations, and generates downloadable PDF reports.

## ▶️ Run Locally

Clone the repository

```bash
git clone https://github.com/chambrishp-web/ai-resume-screener.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
